# Захист дисертації PhD в КПІ ім. Ігоря Сікорського

Покроковий гайд, зразки та приклади документів для аспірантів, які готуються до захисту дисертації на здобуття ступеня доктора філософії (PhD) в КПІ ім. Ігоря Сікорського.

> [!WARNING]
> Документи актуальні станом на літо 2025 року. Перед використанням обов'язково перевіряйте актуальність зразків на [сайті Вченої ради КПІ](https://rada.kpi.ua).

---

## З чого почати

1. Прочитай **[Записки молодого аспіранта](guide.md)** — покроковий гайд з поясненнями, термінами, підводними каменями та порадами
2. Ознайомся з **[Основними нормативними документами](normative-docs.md)** — перелік законів, постанов та правил КПІ
3. Завантаж потрібні **[зразки](2%20-%20Зразки%20документів%20для%20PhD%20(з%20сайту%20ВР%20КПІ)/)** та подивись **[приклади](3%20-%20Приклади%20документів%20для%20PhD/)** заповнених документів

---

## Як зробити свій внесок

Цей репозиторій створений для аспірантів, які йдуть слідом. Якщо ти вже захистився або помітив неактуальну інформацію:

- **Застарілий документ?** — Створи [Issue](../../issues) з описом
- **Є свої приклади документів?** — Створи Pull Request
- **Помилка в гайді?** — Виправ та надішли PR
